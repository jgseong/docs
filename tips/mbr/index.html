MBR 제거

Windows

  1. HxD 실행 - HxD 다운로드
  2. 관리자 권한으로 실행
  3. 디스크 열기 (기타설정 - 디스크 열기)
  4. 읽기 전용으로 읽기 체크박스 해제 - 물리디스크 - 해당 디스크 선택
  5. 디스크 수정에 대한 경고 팝업 - '수락'
  6. 블록 선택 (편집 - 블록 선택, Ctrl+E)
  7. '10진수' 선택
  8. 오프셋 시작 0, 길이 446
  9. 0x00000000 ~ 0x000001BD (446 Bytes) : MBR 영역
  10. '수락'
  11. 선택 채우기 (편집 - 선택 채우기)
  12. 프리셋 삭제 방법 - 제로바이트
  13. '수락'

  14. 저장 (파일 - 저장, Ctrl+S)

  15. 파티션의 부팅 가능 속성 제거

  16. '명령 프롬프트' 실행(관리자 권한)
  17. diskpart 실행
  18. 예시) 디스크 3 의 1번 파티션이 부팅 가능 속성을 제거할 경우.

Linux

  • fdisk 또는 disk 관련 유틸을 통해 해당 디스크의 장치 파일을 파악
  • Example) /dev/sda
sudo dd if=/dev/zero of=/dev/sda bs=446 count=1
  • MBR 영역을 0으로 채움
  • bs(block size), count : bs 크기를 count 만큼 수행
